Hardware you'll be needing or may need

The first thing that you will need to do is to purchase the hardware. There are many retailers all over the world that stock and sell the Raspberry Pi, its components, and many other add on hardware that is optional for the tutorials on this website and the hundreds of other project possibilities.

First, you will need the Raspberry Pi. The raspberry Pi is a micro-computer that can be used as a computing device or logic controller on many projects. It has an array of in and out interfaces (I/O) that can be utilized and programmed.

The hard part is knowing which products you will need to start off, or if any packaged products will be of use. There are many models of the Raspberry Pi, the model we will use is the 3rd generation Raspberry Pi 3 B+. We don’t recommend using any model older than this but do recommend that if you are buying a new Raspberry Pi, you purchase one of the new 4th Generation Raspberry Pi 4. The instructions for the tutorials on this site are the same for all models. There are three models in the 4th generation, namely 1GB, 2GB and 4GB. For the tutorials contained here, the 1GB option will suffice, but if you are looking to do more with your Pi outside the scope of the tutorials, then go with one of the beefier options. It may be a more future proof option to go for a beefier one too.

Go to this link to find an approved reseller in your country. Just click ‘Buy Now’ and select your country to see the approved resellers.

The hardware you will need is:

  • The Raspberry Pi 4B computer (board) – 1GB, 2GB or 4GB
  • Raspberry Pi 4 power supply
  • Micro HDMI to HDMI cable (new to 4th Generation)
  • MicroSD card (minimum 8GB)

Note: it is recommended that you also purchase a case for your raspberry pi board. These can be purchased from the same resellers.

The hardware needed for ALL tutorials

  • Solderless breadboard “full size” with 830 tie points
  • Jump wires (minimum 65)
  • 40-Pin GPIO Breakout and Cable
  • Light emitting diodes (LEDs) of different colours
  • Resistors (recommended 470 ohms but as low as 75 ohms or higher will work). For more information on resisters and how to read them, visit All About Circuits
  • Buzzer

Raspberry Pi 4 setup

  1. Make sure the MicroSD card is in the slot with NOOBS or other OS installed.
  2. Plug in ethernet port for a wired connection (optional)
  3. Plug in mouse and keyboard (wireless recievers should work, depending on drivers)
  4. Plug in mini HDMI – plug other end into monitor
  5. Plug USB-C power cable in before connecting to the wall outlet
  6. Turn on monitor, go to the right input, and you should see the splash screen
